## 🧠 Overview

In today’s fast-paced world, maintaining friendships can be challenging.
**KeenKeeper** is built to simplify that — helping users track, manage, and visualize their interactions with friends in an intuitive way.

It transforms scattered social interactions into **organized, meaningful insights**.

---

## ✨ Key Features

* 🎨 **Modern UI/UX** – Clean, responsive, and user-friendly design
* ⚡ **One-Click Interactions** – Quickly manage friend activities
* 📅 **Interaction History** – Keep track of all previous engagements
* 📊 **Data Visualization** – Interactive pie charts for insights
* 🔔 **Toast Notifications** – Instant feedback for actions
